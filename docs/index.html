Install steps

  1. Tap the red Download button above
  2. When download finishes, open the file
  3. If prompted, enable "Install unknown apps" for Chrome
  4. Tap Install
  5. Open the app, grant "All files access"
  6. Connect to your Mac via USB, drop ROMs into Documents/AtariRoms/
